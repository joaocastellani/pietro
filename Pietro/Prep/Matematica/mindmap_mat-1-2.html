Definição e Notação

  • Multiplicação repetida de um fator
  • Base a: fator que se repete
  • Expoente n: número de repetições
  • Potência: resultado da operação
aⁿ = a · a · … · a (n vezes)
Ex: 2⁴ = 16 · 3³ = 27
Não testado
2

Casos Especiais

  • a¹ = a (expoente 1 = própria base)
  • a⁰ = 1 (desde que a ≠ 0)
  • 1ⁿ = 1 (qualquer n natural)
  • 0ⁿ = 0 (n ≠ 0)
  • 0⁰ = indeterminado ⚠
5⁰ = 1 · 0¹² = 0 · 1⁵⁸ = 1
Não testado
3

Sinal — Base Negativa

  • Expoente PAR → resultado positivo
  • Expoente ÍMPAR → resultado negativo
  • (−a)ⁿ: sinal dentro dos parênteses
  • −aⁿ: sinal aplicado após a potência
(−3)⁴ = +81 ≠ −3⁴ = −81
Não testado
4

Expoente Negativo

  • Inverso da potência de exp. positivo
  • Resultado é fração, não negativo!
  • Válido apenas para a ≠ 0
a⁻ᵐ = 1/aᵐ
2⁻³ = 1/8 · (2/3)⁻² = 9/4
Não testado
5

Produto de Mesma Base

  • Mantém a base, soma os expoentes
  • Só funciona com bases IGUAIS
  • Funciona com expoentes negativos
aᵐ · aⁿ = aᵐ⁺ⁿ
5³ · 5⁷ = 5¹⁰
Não testado
6

Divisão de Mesma Base

  • Mantém a base, subtrai os expoentes
  • Resultado pode ser exp. negativo
  • Base ≠ 0 obrigatório
aᵐ / aⁿ = aᵐ⁻ⁿ
10⁸ / 10⁵ = 10³
Não testado
7

Potência de Potência

  • Expoentes se MULTIPLICAM (não somam)
  • Aplicar de dentro para fora
  • Cuidado com confusão com produto
(aᵐ)ⁿ = aᵐⁿ
(2³)⁴ = 2¹² · (3²)⁵ = 3¹⁰
Não testado
8

Potenciação de Fração

  • Elevar numerador E denominador
  • b ≠ 0 obrigatório
  • Válido para exp. negativos também
(a/b)ⁿ = aⁿ / bⁿ
(2/3)⁴ = 16/81
Não testado
9

Notação Científica

  • Forma: N × 10ⁿ, com 1 ≤ N < 10
  • n > 0: número grande · n < 0: pequeno
  • Soma/sub: igualar expoentes 1º
  • Mult: mult. coef., soma exp.
  • Div: div. coef., subtrai exp.
384 000 km = 3,84×10⁵ km
1,0×10⁻⁹ m = nanoescala

💡 Dicas de Ouro — Potenciação

1
Parênteses fazem toda a diferença: (−3)⁴ = +81 porque o sinal está dentro e é elevado. −3⁴ = −81 porque só o 3 é elevado. Sempre observe a posição do sinal!
2
Expoente negativo ≠ resultado negativo: 2⁻³ = 1/8, não −8. O expoente negativo inverte a base (cria fração). O sinal do resultado vem da base, não do expoente.
3
0⁰ é indeterminado, não é 1: A regra a⁰ = 1 exige a ≠ 0. Se a prova apresentar 0⁰, a resposta correta é "indeterminado", nunca 1.
4
Produto de potências só funciona com mesma base: 2³ · 3³ = 6³ (aqui os expoentes são iguais → potência de produto), mas 2³ · 3⁴ não tem atalho. Confirme sempre se as bases são iguais antes de somar expoentes.
5
Notação científica: verificar o coeficiente após operar: 18 × 10¹¹ não é notação científica correta. Sempre ajustar: 18 × 10¹¹ = 1,8 × 10¹². Na adição, igualar os expoentes antes de somar os coeficientes.
6
Potência de potência: multiplicar, não somar: (a²)³ = a⁶ (multiplicou 2×3). Não confundir com produto a² · a³ = a⁵ (somou 2+3). Macete: produto → soma · potência de potência → produto.
