a. ( ) 0,4 cm/kg

d. ( ) 20 cm/kg
e. ( ) 40 cm/kg
Alternativas:
  a. ( ) 0,4 cm/kg
  b. ( ) 2,5 cm/kg
  c. ( ) 8 cm/kg
  d. ( ) 20 cm/kg
  e. ( ) 40 cm/kg

## PADRÃO DAS QUESTÕES
- Estilo predominante: misto
- Foco: cálculo algébrico com radicais, simplificação, propriedades operacionais e aplicação contextualizada
- Nível de dificuldade médio: médio
- Tópicos mais cobrados: simplificação de radicais, operações com radicais (soma, produto, divisão), potências de expoente fracionário, racionalização de denominadores, aplicação de radicais em contextos reais (queda livre, velocidade do som, IMC)
- Total: 16 questões do capítulo + 2 questões de concurso
